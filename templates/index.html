College Students Match

Following is a sample of the csv format that must be provided for results

  • Row 1 must contain index numbers starting from 1
  • Row 2 must contain names of the participating colleges
  • Row 3 must contain names of the participating students
  • Row 4 must contain number of the participating colleges and students
    in the format N=No. of students and colleges
  • Row 5 must contain the heading of the preferance list being followed:
  • Row 6 onwards there must be the Students Preferance List in the following format
    • Column 1 must contain the college name
    • Column 2 onwards must contain the Students Preferance List of the particular college
  • The Universities Preferance List by the students must be in the following format
    • Column 1 must contain the student name
    • Column 2 onwards must contain the Universities Preferance List
  • Following the row where the Students Preferance List ends must contain a row with the heading of the preferancelist being followed

CSV File in any other format will lead to error
